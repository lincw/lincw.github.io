Situations
...

  1. too many tags
    1. make search difficult
  2. inconsistent tag names
  3. management difficult

plugins
...

  1. TagFolder 以聯集的方式將各種標籤組合在一起，便於搜尋時找到相關的筆記。

    特點

    • 以_untagged 將未設置標籤的筆記收集在一起，便於日後重新整理。
    • open scroll view 功能可以將設置有同一籤標的筆記一併顯示於獨立分頁，便於檢視。
  2. TagWrangler

    特點

    • 針對既有的標籤進行管理，包含 重新命名、搜尋 等

Solutions
...

  1. using multiple levels tag, e.g., '#travel/asia', '#travel/europe'.
  2. using pin function in TagFolder to show the most frequent used tags.
