查詢語法
...

  • 完成 / 未完成：done 或 not done
    • 完成日期：done before/after/on <date>
  • 無到期日：no due date
  • 到期日過濾：due before/after/on <date>
    • 日期可使用 today, yesterday, tomorrow, next week, last Friday, in two weeks, ...
  • 路徑：
    • 包含於搜尋：path includes <path>
    • 排除於搜尋：path does not include <path>
  • 事項描述
    • description includes <string>
    • description does not include <string>
  • 最靠近之標題
    • heading includes <header>
    • heading does not include <header>
  • 是否為重覆待辦事項
    • is recurring
    • is not recurring
  • 排除指定事項：excludes <task>
  • 限制顯示事項查詢數目：limit to <numeric> tasks
  • 排序：sort by status/due/done/path/description
  • 顯示樣式隱藏 hide：
    • edit button
    • backlink
    • done date
    • due date
    • recurrence rule
    • task count
